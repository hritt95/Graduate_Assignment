Dom Manipulation

Use CSS and jquery to dynamically update the DOM

  1. Add a hover effect to the class tab. You will know its working if Facemasks and toner change to plum when you hover over them.
  2. Using jquery remove the active class form any element that currently has it. Then add the active class to element that you clicked on. You know it worked if when you click on of the tabs that tab remains plum color.
  3. Using jquery remove the show class that any element that has it. Then add the show class to the content element that matches to the tab you clicked. You know it works that your content updates when you click the tab.
